# 🔗 API Test Cases

## ✅ What's Inside:
- Methods tested: GET, POST (with sample payloads)
- Format: Excel (`.xlsx`)
- Covers both positive and negative scenarios

These tests simulate basic API contract and response validations for beginner QA work.
